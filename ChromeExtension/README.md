# Lead Tracker Chrome Extension

A simple Chrome Extension to track and save your leads.

## How to Install and Run

Since this is a Chrome Extension, it runs directly in your browser rather than through a local development server. Follow these steps to load it:

1. Clone or download this repository to your local machine.
2. Open Google Chrome and type `chrome://extensions/` into your address bar.
3. Enable **Developer mode** using the toggle switch in the top right corner.
4. Click the **Load unpacked** button that appears in the top left.
5. Select the `ChromeExtension` folder from your downloaded repository.

The extension is now installed and ready to use! You can pin it to your browser toolbar for easy access.
